# Sales Performance Analytics Dashboard

## Project Overview

This project analyzes Superstore sales data from 2021–2024 using Python and Power BI.

The objective was to identify sales trends, profitability drivers, customer segments, and the impact of discounts on profit.

## Key Insights

- Technology category generated the highest sales.
- East region produced the highest profit.
- Consumer segment contributed the largest share of profit.
- Standard Class dominated shipping mode sales.
- Sales peaked in Q4, especially December.
- Higher discounts significantly reduced profitability.
